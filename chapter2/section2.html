2.splitter

在左侧目录和右侧内容之间添加一个可以拖拽的栏，用来调整两边的宽度。

<----------------

3.github

在右上角显示 github 仓库的图标链接。

4.disqus

添加 disqus 评论插件。

{
    "plugins": ["disqus"],
    "pluginsConfig": {
        "disqus": {
            "shortName": "XXXXXXX"
        }
    }
}

如果想要某页面禁用评论可以在文件最前面添加：

---
disqus: false
---
